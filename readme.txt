Adds options to enable Google Optimize integration with your favorite Google Analytics plugins for WordPress.

== Description ==

This adds the needed changes for your site to use the Google Optimize service. This does require setting up an account at optimize.google.com. Currently in public testing, but may be limited number of users accepted.

Works with:

+ [Google Analytics for WordPress by MonsterInsights](https://wordpress.org/plugins/google-analytics-for-wordpress/)
+ [Google Analytics by Analytify](https://wordpress.org/plugins/wp-analytify/)
